# Windows 11 Standalone ZIP Decryptor & Extractor (win11_zip_decryptor)

## 繁體中文

本專案提供一個獨立且輕量化的工具，用於自動解析特定檔名格式之加密 ZIP 檔案（例如由 Windows 11 壓縮工具產生的 ZipCrypto 加密檔案）並完成解密與解壓縮。

### 🌟 特色

1. **零依賴雙擊執行 (Windows)**：`extract_zips.bat` 是一個 Batch + PowerShell + C# 的 Polyglot 檔案。在 Windows 環境下，直接雙擊即可自動在記憶體中編譯並執行 C# 解密程式，無需安裝任何 .NET SDK 或第三方解壓縮工具。
2. **自動密碼生成**：針對 `xxx-D-xxxxx-[段落].[序號].zip` 格式（如 `115-D-02190-00046.1.zip`），工具會自動提取段落號碼（如 `00046`）並拼接前置 `000` 生成正確解密密碼（如 `00000046`）。
3. **支援 Data Descriptor (數據描述符)**：完美支援 Windows 11 壓縮檔常見的 Data Descriptor 結構 (`gp_flag` bit 3)，修復了傳統循序掃描工具在解壓多檔案時會提前中斷的 Bug。
4. **跨平台驗證**：提供 Python 版本 (`verify_zipcrypto.py`) 方便在 Linux 或 Mac 環境進行演算法調試與密碼校驗。

### 📂 檔案清單

* `ZipDecryptor.cs`：C# 實現的 ZipCrypto 解密與解壓縮核心程式碼。
* `extract_zips.bat`：Windows 雙擊自動編譯執行的批次檔（內嵌 C# 原始碼）。
* `verify_zipcrypto.py`：Python 實現的 ZipCrypto 循序解密與驗證工具。
* `test_runner.py`：自動化測試套件，用於產生多檔案加密 ZIP 檔案並測試解密功能。

### 🚀 使用說明

#### 在 Windows 上執行（最簡便）
1. 將 `extract_zips.bat` 置於包含加密 `.zip` 檔案的資料夾中。
2. **雙擊 `extract_zips.bat`** 執行。
3. 程式將自動掃描當前目錄、生成密碼、解密並將檔案解壓至同名資料夾中。

#### 在 Linux / macOS 上使用 Python 驗證
1. 確保已安裝 Python 3。
2. 執行驗證工具：
   ```bash
   python3 verify_zipcrypto.py
   ```

#### 執行整合測試
若要驗證解密核心對單檔案及多檔案（含 Data Descriptor）的支援：
```bash
python3 test_runner.py
```

## English

A lightweight, standalone tool to automatically parse, decrypt, and extract ZIP archives encrypted with standard ZipCrypto (commonly generated by Windows 11 built-in compression utility).

### 🌟 Key Features

1. **Zero-Dependency Double-Click (Windows)**: `extract_zips.bat` is a polyglot Batch + PowerShell + C# script. On Windows, simply double-click it to dynamically compile and run the C# decryptor in memory. No external SDKs or tools required.
2. **Auto Password Generation**: Automatically extracts the segment number from filenames matching `xxx-D-xxxxx-[segment].[index].zip` (e.g., `115-D-02190-00046.1.zip` -> segment `00046`) and prepends `000` to generate the password (e.g., `00000046`).
3. **Data Descriptor Support**: Correctly parses and skips the Data Descriptor blocks (`gp_flag` bit 3) commonly present in Windows 11 ZIP archives, solving the early-termination bug when extracting multi-file archives.
4. **Cross-Platform Verification**: Includes a Python validator (`verify_zipcrypto.py`) for easy testing and debugging on Linux and macOS.

### 📂 File Structure

* `ZipDecryptor.cs`: Core C# ZipCrypto decryption and extraction implementation.
* `extract_zips.bat`: Windows batch runner that compiles and executes the C# decryptor on-the-fly.
* `verify_zipcrypto.py`: Python validator for ZipCrypto header checking and decryption.
* `test_runner.py`: Automated test suite that creates temporary multi-file encrypted ZIP archives to verify the extraction logic.

### 🚀 Usage

#### Run on Windows (Easiest)
1. Place `extract_zips.bat` in the same directory as your encrypted `.zip` files.
2. **Double-click `extract_zips.bat`** to run.
3. The script will automatically scan the folder, derive the password, decrypt, and extract the files to a folder of the same name.

#### Run Python Validator on Linux / macOS
1. Ensure Python 3 is installed.
2. Run the validator:
   ```bash
   python3 verify_zipcrypto.py
   ```

#### Run Automated Tests
Verify both single-file and multi-file (with Data Descriptor blocks) extraction logic:
```bash
python3 test_runner.py
```
